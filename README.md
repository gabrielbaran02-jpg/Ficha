# DiceFlow

Site moderno para rolar dados de RPG (D4, D6, D8, D10, D12, D20, D100 e customizados), com foco em visual e fluidez:

- D20 em SVG (imagem vetorial) no centro.
- Animação avançada de rolagem com spin + troca rápida de números.
- Rolagem única e em lote (x10).
- Histórico com até 12 entradas.
- Estatísticas da sessão (último total, média, críticos).
- Tema escuro/claro.

## Como rodar

Abra `index.html` no navegador.
