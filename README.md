# DiceFlow

Site simples e bonito para rolar dados de RPG (D4, D6, D8, D10, D12, D20, D100 e customizados), com:

- Animação no dado.
- Rolagem única e em lote (x10).
- Histórico com até 12 entradas.
- Estatísticas da sessão (último total, média, críticos).
- Tema escuro/claro.

## Como rodar

Basta abrir o `index.html` no navegador.
